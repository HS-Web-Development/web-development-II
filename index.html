Web Development II - 2022/23 Semester 2

Course Description

This course is a continuation of Web Development I. It builds on existing HTML & CSS skills with JavaScript to allow students to add interactivity into web pages. Students will also be introduced to the concept of mobile-first website, through the Bootstrap framework. There will be opportunities for students to gain practical experience coding up webpages, as well as working in a team environment for a project. The concept of version control with git will also be covered in the course.

List of Topics

  • Version Control & Git
  • JavaScript
  • Mobile-First Design
  • Web Frameworks & Bootstrap
  • Working in a Collaborative Environment
